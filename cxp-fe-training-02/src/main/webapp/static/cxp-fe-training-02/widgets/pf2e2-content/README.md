# Backbase Training Exercise## Portal Frontend - Module 2### Exercise 2_note: This exercise is 3 of 4 in a series of exercises that follow on from each other and should not be used independently__note: All exercises use the [standard portal set-up for backbase trainings](https://my.backbase.com/resources/how-to-guides/getting-your-first-launchpad-based-portal-set-up/)_#### DescriptionIn this exercise, you will learn how to extend the default ICE widget by creating your own ICE templates.#### Steps - Create a new ICE widget by copying the default Backbase ICE widget and remove the link from the template "article-left.html" - Create a template named "media-object.html" and add it in the ICE widget from the previous step - Use Twitter Bootstrap styling#### Additional resources - [server-catalog-widgets.xml](../../../../../config-info/import/server-catalog-widgets.xml#L53-L77)#### References - [Create an ICE Widget](https://my.backbase.com/resources/documentation/portal/5.5.1.0/devd_tuto_ice_.html) - [Add a Content Template](https://my.backbase.com/resources/documentation/portal/5.5.1.0/devd_mang_icet_reftemp.html) - [Media Object](http://getbootstrap.com/components/#media) - [Twitter Bootstrap](http://getbootstrap.com/)
